Experiment Log

This document records the progression of the machine learning and deep learning experiments conducted throughout this project. Each experiment modifies one component of the modelling pipeline while keeping the remaining configuration unchanged wherever possible. This controlled experimental design allows performance differences to be attributed directly to specific modelling decisions, improving both reproducibility and the interpretability of the experimental results.

The dataset was split chronologically, using observations from 2017–2022 for training and 2023–2025 for testing to preserve the temporal structure of the forecasting task and prevent information leakage. Because the target variable is highly imbalanced (approximately 87.7% Crisis and 12.3% No Crisis), ROC-AUC was used as the primary evaluation metric, while accuracy, precision, recall, and F1-score were reported as complementary measures.

Overall Findings

Across all experiments, ROC-AUC was prioritised as the primary evaluation metric because it provides a more reliable measure of model discrimination than overall accuracy for highly imbalanced classification problems. Although several classical machine learning models achieved accuracies close to 89%, their confusion matrices showed poor minority-class performance, demonstrating that accuracy substantially overestimated their practical usefulness.

Among the classical machine learning approaches, the Balanced Random Forest provided the most balanced predictions by improving minority-class detection through class weighting. Among the deep learning approaches, the Regularized Sequential Neural Network achieved the strongest overall generalization, maintaining a validation ROC-AUC of approximately 0.80 while reducing overfitting through Batch Normalization, Dropout, and Early Stopping.

Overall, the experiments show that careful preprocessing, feature engineering, class balancing, and regularization contributed more to predictive performance than increasing model complexity alone, supporting the use of conflict event data as a meaningful short-term early-warning signal for anticipating food insecurity.
